## What I'm Building

**[Big Chungus Cluster (BCC)](https://github.com/404akshat/big-chungus-cluster)** : a self-hosted Kubernetes PaaS written in Go.  
You give it a Docker image. It handles deployments, scaling, observability, chaos engineering, and security.  
All on K3s. All from scratch. 21 components. One week at a time.

| Component | Status |
|-----------|--------|
| Core API + PostgreSQL + bcc-ctl CLI | ✅ Done |
| Terraform + Helm | 🔨 In Progress |
| Observability (Prometheus / Grafana / Loki) | 📋 Planned |
| Chaos Engineering | 📋 Planned |
| Security (Trivy / Falco / Vault) | 📋 Planned |

> Week 1: *"I hit my first hardware bottleneck. It wasn't a cloud limit — it was the physical melting point of my desk."*
